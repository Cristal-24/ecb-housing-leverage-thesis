# Replication Materials

This repository contains the replication materials for my Master 2 thesis:

**Household Leverage and the Heterogeneous Transmission of European Central Bank (ECB) Monetary Policy Shocks to Euro-Area Housing Markets**

The folder includes the raw data, cleaned datasets, Python scripts, regression outputs, and figures used for the empirical analysis.

The empirical analysis studies whether household leverage amplifies the response of euro-area housing markets to European Central Bank monetary policy shocks. The analysis includes baseline fixed-effects regressions, local projections, and a robustness check using mortgage-specific leverage.

The scripts are numbered in the order in which they were used.
